HomeBrew，简称 brew ，是 macOS 下的软件安装程序，下面记录几个常用的命令
```
    1. 安装工具
    举个栗子:   wget  这个工具
    输入命令:   brew install wget 
    
    2. 卸载
    拿上面的栗子:   wget 工具
    输入命令:    brew uninstall wget
    
    3. 其他命令
    brew list       列出已安装的软件
    
    brew update     更新brew
    
    brew home       用浏览器打开brew的官方网站
    
    brew info       显示软件信息
    
    brew deps       显示包依赖

```
